# doc-indexer

Document Indexer is a web service.

Supports:

- Managing Chrome bookmark file.

Requirements:

- MySQL
- Consul
- RabbitMQ
- Consul
- [GoAuth](https://github.com/curtisnewbie/goauth)


## Configuration

| Property             | Description         | Default Value   |
|----------------------|---------------------|-----------------|
| docindexer.temp-path | Temporary file path | /tmp/docindexer |
